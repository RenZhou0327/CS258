——Naive的数据压缩

​	这是我们小组自己实现的压缩/解压文件软件，说明如下：

​	——dist文件夹

​		内有可执行程序mainUI.exe，点击即可运行

​		mainUI.py是主界面源程序，主要调用tkinter第三方库设置图形化界面

​		HuffmanKernel.py，LzssKernel.py是两种压缩/解压文件方式实现的内核程序，里面没有调用任何zip

​		gzip等python中本地的压缩文件库或第三方库。

​		我们的实现软件仅仅旨在更深入地了解如何将理论上的算法实际地应用在压缩/解压文件的操作中，因此是Naive的软件，
		在性能上和可靠性上都与成熟的压缩软件有较大差距，因而希望老师和助教仅仅做简单体验即可，请不要做极端测试，
		请不要对重要文件进行测试， 使用前可以观看我们在PPT中的演示视频。

​	——实验文件

​		里面包含我们测试的一些文件，均可以使用两种算法成功压缩和解压。
